O que é Data Science?

A Ciência de Dados (Data Science, em inglês) é um campo multidisciplinar para apoiar atividades orientadas a dados, integrando e desenvolvendo ideias, métodos e processos para obter informações a partir de dados. Esses métodos advêm de diferentes áreas, principalmente da Estatística e da Ciência da Computação, e, quando unificados, dão origem à Ciência de Dados. Técnicas de modelagem estatística, classificação, identificação de padrões e de tendências e mineração de dados são alguns meios da área de Ciência de Dados para a análise intensiva de diferentes tipos de dados e geração de informações para a tomada de decisões.
